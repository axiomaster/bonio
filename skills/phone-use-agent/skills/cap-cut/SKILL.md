# 剪映 (CapCut) - Video Editing

剪映是HarmonyOS上首选的视频编辑应用，用于创建和编辑短视频。

## When to Use

**Use this skill when:**
- 需要在手机上完成视频剪辑
- 需要创建短视频、添加音乐、特效
- 需要使用模板快速制作视频
- 需要将视频发布到抖音

**剪映是视频剪辑的首选应用**

## Recommended Approach: 剪同款 (Templates)

**优先使用"剪同款"功能** - 更简单、效果更好、成功率更高。

### 剪同款使用流程

1. 点击底部第2个tab **"剪同款"**
2. 在搜索框输入主题关键词（如"春节"、"旅行"、"生日"等）
3. 浏览并选择喜欢的模板
4. 点击模板上的 **"剪同款"** 按钮
5. 选择素材（见下方"系统Picker选择说明"）
6. 等待自动生成视频
7. 导出并分享

### 素材选择建议

| 模板类型 | 建议素材风格 |
|----------|--------------|
| 春节模板 | 红色、喜庆、团圆氛围 |
| 旅行模板 | 风景、人物、地标 |
| 生日模板 | 蜡烛、蛋糕、欢乐场景 |
| 婚礼模板 | 浪漫、温馨、白色/粉色 |

**重要**: 选择与模板风格相近的素材，效果更好

## System Picker 选择说明

选择素材时会打开**系统图片Picker**，界面为宫格布局：

| 元素 | 说明 |
|------|------|
| 宫格布局 | 照片/视频以网格形式排列 |
| 选中标记 | 每个宫格**右下角的小圆点** |
| 选择操作 | **点击小圆点**选中对应内容 |
| 已选中状态 | 小圆点变为勾选标记 |

**注意**: 点击宫格本身是预览，**必须点击右下角小圆点才能选中**

## Main Editing Features (剪辑 Tab) - 备用方案

如果"剪同款"不适用，可使用"剪辑"tab的功能：

### 1. 开始创作
- 点击顶部"开始创作"
- 手动选择照片或视频素材
- 可以进行精细剪辑、添加音乐、特效等
- 适合需要自定义编辑的场景

### 2. 一键成片
- 点击顶部"一键成片"
- 选择照片或视频素材（建议不超过50张）
- 系统自动生成视频

## Export and Share Workflow

### 方式一：直接分享到抖音（推荐）

```
剪辑完成 → 点击"导出" → 等待渲染 → 点击"分享" → 选择"抖音" → 填写描述/文案 → 点击"发作品"
```

1. 视频剪辑完成后，点击右上角**"导出"**
2. 等待视频自动渲染生成
3. 渲染完成后，点击**"分享"**
4. 选择分享到**"抖音"**
5. 填入视频描述、文案、标签等
6. 点击**"发作品"**完成发布

### 方式二：保存到图库

```
剪辑完成 → 点击"导出" → 等待渲染 → 点击"完成" → 视频保存到图库
```

1. 渲染完成后，点击右上角**"完成"**
2. 视频自动保存到图库
3. 后续可通过图库再次分享或发表

## Tips

1. **优先使用剪同款**: 模板效果更好、成功率更高
2. **搜索关键词**: 在剪同款页面搜索主题词（如"春节"、"旅行"）找模板
3. **系统Picker选片**: 必须点击宫格**右下角小圆点**选中素材，点击宫格本身是预览
4. **素材风格匹配**: 选择与模板风格相近的素材效果更好
5. **素材数量**: 一键成片最多支持50张照片，剪同款按模板要求
6. **直接分享**: 导出后选择"分享"可直接发布到抖音

## Limitations

- 一键成片最多支持50张照片
- 剪同款需要按照模板要求的素材数量
- 渲染时间取决于视频长度和分辨率
- 部分高级模板可能需要会员
