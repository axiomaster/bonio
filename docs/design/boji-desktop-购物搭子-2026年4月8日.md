# 功能定义：BoJi 桌面助手 —— “搜同款”

### 1. 交互流程
- **右键菜单**：用户点击 BoJi 右键，选择“搜同款”。
- **圈选动作**：鼠标变更为十字星，用户拖拽选择屏幕上的物体。
- **即时截图**：松开鼠标，系统自动截取圈选区域。
- **Avatar 响应**：BoJi 实体移动到圈选区下方，播放搜索动画。
- **结果弹出**：BoJi 下方直接弹出浏览器（WebView），自动跳转至淘宝搜同款结果页。

### 2. 状态机表现
- **待命**：普通 Idle 状态。
- **圈选时**：BoJi 变更为 Alert 状态，关注用户选区。
- **搜索时**：Thinking 动画，头顶冒出搜索气泡。
- **展示时**：指向弹出的浏览器窗口，切换为 Speaking 或 Happy 状态。

### 3. 核心优势
- **低成本**：无需购买昂贵的商业搜图 API。
- **通用性**：利用淘宝网页版能力，覆盖全量商品数据。
- **拟人化**：通过 BoJi 的位移和引导，让一次简单的搜索变成“宠物帮忙找东西”的有趣体验。
