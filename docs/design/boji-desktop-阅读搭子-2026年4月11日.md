# 产品需求文档 (PRD)：波妞 (Bonio) 桌面助手 —— 深度效能版

**产品名称:** 波妞 (Bonio)
**文档版本:** V6.0
**核心目标:** 通过伴读、转录及实时字幕三大核心功能，将波妞从“桌面陪玩”升级为“深度学习与会议辅助智能体”，并构建基于 Memory 系统的知识闭环。

---

## 1. 功能一：波妞伴读 (Bonio Reading)
**场景描述：** 当用户在阅读长篇技术文档、深度深度报道或专业论文时，波妞提供侧边栏式的辅助环境，帮助提炼与记录。

### 1.1 布局与触发 (70/30 分屏)
* **自动布局：** 启动伴读后，波妞强制调整当前阅读窗口（如浏览器、PDF）宽度至屏幕的 **70%**；在屏幕剩余 **30%** 区域弹出【波妞伴读窗口】。
* **触发方式：** * 右键菜单选择 [开始伴读]。
    * 智能感知：当监测到当前活动窗口文本量巨大时，波妞气泡提示询问是否开启。

### 1.2 伴读窗口功能 (Markdown 驱动)
* **智能索引：** 自动提取原文目录，支持点击标题让原文窗口同步滚动。
* **内容生成：** 自动生成文章摘要及段落解析。
* **交互编辑：** 伴读区域采用 Markdown 格式，AI 预填段落摘要，用户可在 Markdown 块内直接进行笔记编辑、增删。
* **物理形态：** 波妞 Avatar 悬浮于 30% 伴读窗口的顶端，呈阅读/记录动作。

### 1.3 存储与记忆 (Memory)
* **保存操作：** 点击“入库”后，将原文链接/内容、AI 摘要、用户笔记一并存入 Memory。
* **标签管理：** 自动打上 `#伴读` 标签。

---

## 2. 功能二：会议转录 (Bonio Minutes)
**场景描述：** 在音视频会议或观看在线课程时，实时将语音转化为结构化文字纪要。

### 2.1 音频截获逻辑
* **系统音流：** 波妞直接截取 PC 内部音频流（Loopback 机制），确保即使不外放声音也能精准转录。
* **实时展示：** 弹出半透明转录面板，实时流式显示文本内容。

### 2.2 模板化摘要系统
转录结束后，波妞根据内容主题引导用户选择【总结模板】：
* **会议摘要：** 自动提取议题、参与人意见、To-do List（待办事项）。
* **赛事摘要：** 提取关键得分点、胜负时刻、精彩表现。
* **课程摘要：** 提取核心公式、概念定义、老师强调的重难点。

### 2.3 存储逻辑
* **结果入库：** 结构化纪要存入 Memory 系统。
* **标签管理：** 自动打上 `#转录` 标签。

---

## 3. 功能三：双语字幕 (Bonio Captions)
**场景描述：** 实时翻译当前 PC 发出的任何音频内容，解决跨语言沟通障碍。

### 3.1 视觉表现
* **UI 位置：** 采用双行悬浮设计。位置固定在转录窗口上方，波妞 (Bonio) Avatar 的正下方。
* **显示逻辑：** * **第一行：** 原始音频转录文本（如英文）。
    * **第二行：** 实时翻译结果（如中文）。
* **动态响应：** 无音频时自动隐身，保持桌面清爽。

### 3.2 性能要求
* **翻译延迟：** 需控制在 500ms 以内，确保声画同步感。

---

## 4. 实体交互状态机 (State Machine) 更新

| 状态名称 | 视觉动画 (Lottie/Spine) | 场景关联 |
| :--- | :--- | :--- |
| **READING** | 拿笔记录、翻书、戴眼镜、偶尔抬头思索。 | 伴读模式开启中。 |
| **LISTENING** | 戴上耳机，身体随声波轻微律动。 | 转录或字幕功能开启中。 |
| **MEMORIZING** | 做出“向大脑塞入光球”或“整理背包”的动作。 | 伴读/转录结果存入 Memory 时。 |
| **SEARCHING** | 翻找动作，头顶冒出小雷达扫描。 | 用户询问之前记录的内容时。 |

---

## 5. 闭环调取逻辑 (Bonio Memory Recall)

所有的记录最终通过波妞的对话框（双击唤起）进行智能检索：
* **用户输入：** “波妞，帮我查一下上次开会提到的关于产品定价的笔记。”
* **处理逻辑：** 波妞检索 `tag:#转录` 且内容包含 `定价` 的记录。
* **结果展示：** 波妞以气泡卡片形式展示结论，并支持一键打开原始转录面板查看完整版。

---

## 6. 待实施技术点确认

1. **分屏算法：** 针对 Windows/macOS 开发跨平台的窗口重排组件，实现 70/30 布局切换。
2. **音频环回 (Loopback)：** 开发基于系统层级的音频抓取模块（Windows 侧重虚拟声卡，macOS 侧重 Audio Kernel）。
3. **Markdown 编辑器：** 在波妞前端容器中集成轻量化 MD 编辑器，支持即时渲染。
4. **分类器：** 训练/调优小型摘要模型，识别文本意图以匹配对应的摘要模板。
